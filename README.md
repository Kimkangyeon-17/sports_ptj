# Sports PTJ

## 프로젝트 소개
AI를 활용하여 축구 경기를 분석하고 예측하는 종합 스포츠 플랫폼입니다.
관심 팀을 등록하면 맞춤형 대시보드로 경기 정보를 제공받을 수 있습니다.

## ✨ 주요 기능
### 🔐 회원 관리
기본 회원가입/로그인
소셜 로그인 (카카오, 네이버, 구글)
마이페이지 (프로필, 내가 쓴 댓글)

### ⭐ 관심 팀 맞춤 서비스
관심 팀 등록/삭제
맞춤 대시보드 (관심 팀 경기만 표시)
일정표 필터 (내 팀만 보기)

### 🔍 데이터 검색
선수 검색: 선수 정보, 시즌 스탯
팀 검색: 팀 정보, 선수 명단
감독/코치 검색 : 감독, 코치 정보, 스타일 등

### 📊 리그 정보
순위표: 프리미어리그/라리가/분데스리가 + AI 트렌드 분석
일정표: 경기 일정 + AI 승률 예측

### 🤖 AI 경기 분석 (핵심)
경기 프리뷰: AI 승률 예측, 예상 스코어, 핵심 포인트 5가지
경기 리뷰: 경기 통계, AI 종합 평가, MVP 선정
